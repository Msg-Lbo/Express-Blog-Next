## Express Blog Next
### 该项目基于Vite + Vue3 + TypeScript 开发的博客系统
#### 整个项目使用到的技术：
#### 前端：
- Vite
- Vue3
- TypeScript
- Naive UI
- Axios
- Tailwind CSS
#### 后端：
- Express
- Mysql

### 项目运行
>pnpm(推荐)
``` bash
# 安装依赖
pnpm install
# 开发环境
pnpm run dev
# 生产环境
pnpm run build
```
>npm
``` bash
# 安装依赖
npm install
# 开发环境
npm run dev
# 生产环境
npm run build
```
