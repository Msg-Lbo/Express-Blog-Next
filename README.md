## 项目运行
>pnpm(推荐)
``` bash
# 安装依赖
pnpm install

# 开发环境
pnpm run dev

# 生产环境
pnpm run build
```
>npm
``` bash
# 安装依赖
npm install

# 开发环境
npm run dev

# 生产环境
npm run build
```
